# Lijing's Shop — 商品展示网站

基于纯静态技术栈 + Sveltia CMS 无头 CMS + Cloudflare Pages 的商品管理展示网站。

## 功能

- 前台：响应式商品卡片网格展示，支持搜索过滤、懒加载
- 后台：通过 `/admin` 访问 Sveltia CMS 管理商品（增删改查）
- 部署：GitHub -> Cloudflare Pages 自动构建部署

## 技术栈

- HTML + CSS + JavaScript（纯静态，无框架）
- Sveltia CMS（轻量级无头 CMS，<300KB）
- GitHub（代码托管 + 数据存储）
- Cloudflare Pages（部署托管）
